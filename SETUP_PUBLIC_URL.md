# הגדרת כתובת ציבורית ל-RSVP

## הבעיה:
הקישורים ב-WhatsApp מצביעים ל-`http://localhost:3000` - זה לא עובד למוזמנים כי זה רק על המחשב שלך.

## פתרונות:

### פתרון 1: שימוש ב-ngrok (הכי קל - לבדיקה)

**ngrok** מאפשר לחשוף את השרת המקומי לאינטרנט.

1. **הורד ngrok:**
   - https://ngrok.com/download
   - או: `choco install ngrok` (אם יש לך Chocolatey)

2. **הפעל ngrok:**
   ```bash
   ngrok http 3000
   ```

3. **תקבל כתובת כמו:**
   ```
   https://abc123.ngrok.io
   ```

4. **צור קובץ `.env` בתיקיית הפרויקט:**
   ```
   BASE_URL=https://abc123.ngrok.io
   PORT=5000
   ```

5. **הפעל מחדש את השרת**

**⚠️ הערה:** הכתובת משתנה כל פעם שסוגרים ngrok (בגרסה החינמית). לכתובת קבועה צריך חשבון בתשלום.

---

### פתרון 2: שרת אמיתי (לשימוש קבוע)

אם יש לך שרת או VPS:

1. **העלה את האתר לשרת**

2. **צור קובץ `.env`:**
   ```
   BASE_URL=https://yourdomain.com
   PORT=5000
   ```

---

### פתרון 3: שימוש ב-IP הציבורי (רק ברשת מקומית)

אם כל המוזמנים באותה רשת WiFi:

1. **מצא את ה-IP שלך:**
   ```powershell
   ipconfig
   ```
   חפש `IPv4 Address` - למשל: `192.168.1.100`

2. **צור קובץ `.env`:**
   ```
   BASE_URL=http://192.168.1.100:3000
   PORT=5000
   ```

3. **ודא שהח firewall מאפשר חיבורים:**
   ```powershell
   # Allow port 3000
   netsh advfirewall firewall add rule name="Node.js" dir=in action=allow protocol=TCP localport=3000
   ```

**⚠️ זה עובד רק אם המוזמנים באותה רשת!**

---

### פתרון 4: שירותי אירוח (לשימוש קבוע)

- **Vercel** - חינמי ל-Next.js
- **Heroku** - חינמי (מוגבל)
- **Railway** - חינמי
- **DigitalOcean** - בתשלום

## איך להגדיר:

### שלב 1: צור קובץ `.env`

בתיקיית הפרויקט הראשית (איפה שיש `package.json`):

```
BASE_URL=https://your-url.com
PORT=5000
```

### שלב 2: הפעל מחדש את השרת

```bash
npm run dev
```

### שלב 3: בדוק

שלח הזמנה לעצמך ובדוק שהקישור נכון.

---

## דוגמאות לקובץ .env:

### עם ngrok:
```
BASE_URL=https://abc123.ngrok.io
PORT=5000
```

### עם שרת אמיתי:
```
BASE_URL=https://wedding.yourdomain.com
PORT=5000
```

### עם IP מקומי:
```
BASE_URL=http://192.168.1.100:3000
PORT=5000
```

---

## איך לבדוק שהכל עובד:

1. שלח הזמנה לעצמך
2. לחץ על הקישור
3. ודא שהדף נטען

---

## טיפים:

1. **לבדיקה מהירה:** השתמש ב-ngrok
2. **לשימוש קבוע:** העלה לשרת אמיתי
3. **לעדכון BASE_URL:** שנה את `.env` והפעל מחדש

---

## אם עדיין לא עובד:

1. ודא שקובץ `.env` בתיקייה הנכונה
2. ודא שהשרת הופעל מחדש אחרי השינוי
3. בדוק שהכתובת נגישה (נסה לפתוח בדפדפן)
