# HatonaME - אתר תכנון חתונה

אתר מקיף לניהול תכנון חתונה עם תמיכה בעברית, כולל:
- טעינת מוזמנים מקובץ Excel
- סידורי שולחנות אוטומטיים לפי קטגוריות
- העלאת הזמנות עם טקסט
- שליחת הזמנות ב-WhatsApp
- מערכת RSVP דיגיטלית

## התקנה

### דרישות מוקדמות
- Node.js (גרסה 14 ומעלה)
- npm או yarn

### התקנת תלויות

```bash
# התקנת תלויות לשרת ולקוח
npm run install-all
```

או בנפרד:

```bash
# תלויות שרת
npm install

# תלויות לקוח
cd client
npm install
```

## הפעלה

### הפעלת השרת והלקוח יחד

```bash
npm run dev
```

זה יפעיל:
- שרת backend על פורט 5000
- שרת frontend (Next.js) על פורט 3000

### הפעלה נפרדת

```bash
# שרת בלבד
npm run server

# לקוח בלבד (בטרמינל נפרד)
cd client
npm run dev
```

## שימוש

1. פתח את הדפדפן בכתובת: `http://localhost:3000`

2. **יצירת אירוע חדש:**
   - לחץ על "יצירת אירוע חדש"
   - מלא את פרטי האירוע ושמור

3. **טעינת מוזמנים:**
   - פתח את האירוע
   - בחר בטאב "מוזמנים"
   - גרור קובץ Excel עם העמודות הבאות:
     - `שם` או `שם מלא` - שם המוזמן
     - `קטגוריה` - קטגוריה (למשל: "עבודה שון", "עבודה ליפז")
     - `טלפון` - מספר טלפון
     - `כמות` - כמות מוזמנים (אופציונלי)

4. **סידור שולחנות:**
   - בחר בטאב "שולחנות"
   - הגדר קיבולת שולחן
   - לחץ על "סדר שולחנות אוטומטית"
   - המערכת תסדר את המוזמנים לפי קטגוריות

5. **העלאת הזמנה:**
   - בחר בטאב "הזמנה"
   - לחץ על "עלה/ערוך הזמנה"
   - העלה תמונה והכנס טקסט
   - בחר מיקום הטקסט על התמונה

6. **שליחת הזמנות ב-WhatsApp:**
   - בחר בטאב "WhatsApp"
   - סרוק את ה-QR Code עם WhatsApp שלך (בפעם הראשונה)
   - לחץ על "שלח הזמנות לכל המוזמנים"
   - כל מוזמן יקבל הודעה עם קישור ל-RSVP

7. **RSVP:**
   - המוזמנים יקבלו קישור ב-WhatsApp
   - הם יכולים לאשר/לבטל ולציין כמה אנשים יגיעו
   - התשובות נשמרות אוטומטית במערכת

## מבנה הפרויקט

```
HatonaME/
├── server/              # Backend (Node.js/Express)
│   ├── database/        # מסד נתונים SQLite
│   ├── routes/          # נתיבי API
│   ├── services/        # שירותים (WhatsApp, וכו')
│   └── uploads/         # קבצים שהועלו
├── client/              # Frontend (Next.js/React)
│   ├── src/
│   │   ├── pages/       # דפי האתר
│   │   ├── services/    # שירותי API
│   │   └── styles/      # עיצוב
└── wedding_planner.db  # מסד נתונים SQLite (נוצר אוטומטית)
```

## מסד נתונים

המערכת משתמשת ב-SQLite מקומי. הקובץ `wedding_planner.db` נוצר אוטומטית בפעם הראשונה.

### טבלאות:
- `events` - אירועים
- `guests` - מוזמנים
- `tables` - שולחנות
- `invitations` - הזמנות
- `whatsapp_messages` - הודעות WhatsApp

## פתרון בעיות

### WhatsApp לא מתחבר
- ודא שסרקת את ה-QR Code
- בדוק שהטלפון מחובר לאינטרנט
- נסה להפעיל מחדש את השרת

### קובץ Excel לא נטען
- ודא שהקובץ בפורמט .xlsx או .xls
- ודא שיש עמודות עם השמות הנכונים בעברית
- בדוק את הקונסול לשגיאות

### שגיאות אחרות
- בדוק את הקונסול של השרת והלקוח
- ודא שכל התלויות הותקנו
- ודא שהפורטים 3000 ו-5000 פנויים
