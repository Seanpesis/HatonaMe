# תיקון שגיאת פורט תפוס

## הבעיה:
```
Error: listen EADDRINUSE: address already in use :::5000
```

זה אומר שהפורט 5000 כבר תפוס על ידי תהליך אחר.

## פתרונות:

### פתרון 1: סגור את התהליך הישן (מומלץ)

**ב-Windows PowerShell:**
```powershell
# מצא את התהליך
netstat -ano | findstr :5000

# תראה משהו כמו:
# TCP    0.0.0.0:5000    0.0.0.0:0    LISTENING    12345
# המספר האחרון (12345) הוא ה-PID

# סגור את התהליך
taskkill /PID 12345 /F
```

**או דרך Task Manager:**
1. לחץ Ctrl+Shift+Esc
2. לך לטאב "Details"
3. חפש "node.exe"
4. סגור את כל התהליכים של node.exe

### פתרון 2: שנה פורט

1. צור קובץ `.env` בתיקיית הפרויקט (אם אין):
   ```
   PORT=5001
   ```

2. או שנה ישירות ב-`server/index.js`:
   ```javascript
   const PORT = process.env.PORT || 5001;  // שנה מ-5000 ל-5001
   ```

3. אם שינית פורט, גם עדכן את `client/next.config.js`:
   ```javascript
   destination: 'http://localhost:5001/api/:path*',  // שנה מ-5000 ל-5001
   ```

### פתרון 3: הפעל מחדש את המחשב

זה יסיים את כל התהליכים הישנים.

## איך למנוע את זה בעתיד:

1. **תמיד סגור את השרת נכון:**
   - לחץ Ctrl+C בטרמינל
   - חכה שהשרת ייסגר לפני שסוגר את החלון

2. **אל תפתח כמה עותקים:**
   - ודא שיש רק תהליך אחד שרץ

3. **בדוק לפני הפעלה:**
   ```powershell
   netstat -ano | findstr :5000
   ```
   אם יש תוצאה - סגור את התהליך לפני הפעלה מחדש

## אחרי התיקון:

1. סגור את כל החלונות (Ctrl+C)
2. הפעל מחדש:
   ```bash
   npm run dev
   ```
